Contribution
============

If you want to contribute to this document you can:

-  `Create an issue on
   Github <https://github.com/f0rki/mapping-high-level-constructs-to-llvm-ir/issues>`__
-  Fork and fix it yourself. We're happy to merge any pull requests or
   accept patches.
